# 뽑기 프로그램

간단한 추첨 및 지원자 관리 웹 애플리케이션입니다.

## 기능

- 역할(아르바이트) 추가/삭제
- 지원자 추가/삭제
- 역할별 지원자 관리
- 랜덤 추첨 기능
- 지원자 목록 확인 및 관리

## 사용 방법

1. 역할(아르바이트)을 추가합니다.
2. 지원자를 추가합니다.
3. 지원자가 원하는 역할에 지원할 수 있도록 합니다.
4. '추첨하기' 버튼을 클릭하여 랜덤으로 선발합니다.

## 설치

1. 이 저장소를 클론합니다:
   ```
   git clone [저장소 URL]
   ```
2. 웹 브라우저에서 `index.html` 파일을 엽니다.

## 기술 스택

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Bootstrap 5
